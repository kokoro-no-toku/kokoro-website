This REST API allows you to integrate our Japanese AI mental support service — powered by Google Gemini — into your applications, platforms, and workflows. 日本語・English の両言語に対応しています。All responses are UTF-8 encoded JSON. Base URL: https://kokoro-bot-production.up.railway.app/v1 All API requests must be made over HTTPS. Quick Start cURL example copy# Send a chat message to the AI curl -X POST "https://kokoro-bot-production.up.railway.app/v1/chat" \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "user_id": "user_abc123", "message": "最近、仕事のことで悩んでいます。", "language": "ja" }' Authentication All API requests require a Bearer token in the Authorization header. API keys are generated from your dashboard after subscription. Keep your API key secret. Never expose it in client-side code or public repositories. Retry after 60 seconds.", "retry_after": 60 } } LINE Webhook POST /webhook/line LINE platform → your server This endpoint receives events from the LINE Messaging API. Configure this URL in your LINE Developer Console as your webhook URL. The service automatically processes incoming messages and replies via LINE. LINE will verify your server with a X-Line-Signature header. Signature validation is handled automatically.
