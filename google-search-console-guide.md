## Step 1: Search Console アカウント作成 {#step1}

1. **Google アカウントでログイン**
   - https://search.google.com/search-console/ にアクセス
   - ビジネス用 Google アカウントでサインイン（個人アカウントでも可）

2. **「今すぐ開始」をクリック**

---

## Step 2: プロパティの追加 {#step2}

Search Console には 2 種類のプロパティがあります。

| タイプ | 説明 | ペライチの場合 |
|--------|------|--------------|
| **ドメインプロパティ** | example.com 全体を管理 | ドメイン取得後に推奨 |
| **URL プレフィックスプロパティ** | https://example.com/ など特定 URL | **ペライチ初期段階ではこちら** |

### ペライチでの手順
1. 「URL プレフィックス」を選択
2. ペライチで公開したページの URL を入力
   ```
   例: https://your-site.peraichi.com/
   または独自ドメイン取得後: https://kokoro-no-talk.com/
   ```
3. 「続行」をクリック

## Step 3: 所有権の確認（ペライチ向け） {#step3}

Search Console はサイトの所有者であることを確認する必要があります。

### 方法A: HTML タグ（ペライチで最も簡単）

1. Search Console から「HTML タグ」を選択
2. 以下のようなメタタグが表示されます：
   ```html
   <meta name="google-site-verification" content="XXXXXXXXXXXXXXXX" />
   ```
3. **ペライチ管理画面での設定方法：**
   - ペライチ管理画面 → 「ページ設定」→「SEO 設定」
   - 「head タグに追加するコード」欄にメタタグを貼り付け
   - 保存して公開
4. Search Console で「確認」ボタンをクリック

### 方法B: Google アナリティクス（GA4 連携済みの場合）

1. Google Analytics 4 のトラッキングコードをサイトに設置済みの場合
2. Search Console の確認方法で「Google Analytics」を選択
3. 自動的に確認完了

### 方法C: HTML ファイルのアップロード

1. Search Console が提供する `google[英数字].html` ファイルをダウンロード
2. ペライチのファイルアップロード機能でルートに設置
   - ペライチ管理画面 → 「ファイル管理」からアップロード
3. `https://your-site.peraichi.com/google[英数字].html` にアクセスして確認
4. Search Console で「確認」をクリック

> ⚠️ **注意**: ペライチの無料プランではファイルアップロードが制限される場合があります。方法Aが最も確実です。

## Step 4: サイトマップの送信 {#step4}

サイトマップを送信することで、Google がすべてのページを確実に発見できます。

### 4-1. サイトマップの設置

ペライチでは通常、サイトマップを直接アップロードする必要があります：

1. 提供の `sitemap.xml` を `https://your-domain.com/sitemap.xml` に設置
2. **ペライチの場合**: ファイル管理機能でルートに `sitemap.xml` をアップロード

> ペライチの独自ドメインプランが必要な場合があります。  
> ドメインが決まったら URL を更新してください：
> ```xml
> <!-- sitemap.xml 内の example.com を実際のドメインに置き換え -->
> <loc>https://your-actual-domain.com/</loc>
> ```

### 4-2. Search Console でのサイトマップ送信

1. Search Console 左メニュー → 「サイトマップ」
2. サイトマップ URL を入力：
   ```
   sitemap.xml
   ```
3. 「送信」ボタンをクリック
4. ステータスが「成功しました」になることを確認

## Step 5: インデックス登録リクエスト {#step5}

公開後すぐに Google に認識させるためのリクエストを行います。

### URL インスペクションツールの使い方

1. Search Console 上部の検索バーにトップページ URL を入力
   ```
   https://your-domain.com/
   ```
2. 「インデックス登録をリクエスト」ボタンをクリック
3. 同様に以下のページも繰り返す：
   - トップページ: `/`
   - APIドキュメント: `/api.html`
   - 料金ページ: `/pricing/`

> 📌 Google のインデックス反映には通常 **1〜14 日** かかります。

## Step 6: 構造化データの確認 {#step6}

`index.html` には Schema.org の構造化データが組み込まれています。正しく認識されているか確認します。

### 6-1. リッチリザルトテスト

1. https://search.google.com/test/rich-results にアクセス
2. サイトの URL を入力して「テスト」
3. 以下が検出されれば成功：
   - `SoftwareApplication` — アプリ情報
   - `FAQPage` — よくある質問
   - `Offer` — 料金情報

### 6-2. Search Console での確認

- 左メニュー「拡張」→「リッチリザルト」で表示状況を確認
- エラーが表示された場合は `index.html` の LD+JSON を修正

### 6-3. 確認すべき構造化データ一覧

| スキーマタイプ | 掲載箇所 | 期待される検索表示 |
|--------------|---------|-----------------|
| `SoftwareApplication` | index.html | アプリ評価・価格の表示 |
| `FAQPage` | index.html | FAQ の展開表示 |
| `Offer` | index.html | 料金の直接表示 |
| `WebSite` | index.html | サイトリンク検索ボックス |
| `TechArticle` | api.html | 技術文書としての認識 |

## Step 7: ペライチ特有の追加設定 {#step7}

### robots.txt の確認

ペライチは自動で `robots.txt` を生成します。インデックスが許可されているか確認：

```
# 確認 URL
https://your-site.peraichi.com/robots.txt
```

以下が含まれていれば問題なし：
```
User-agent: *
Allow: /
Sitemap: https://your-domain.com/sitemap.xml
```

### ペライチ SEO 設定のチェック

ペライチ管理画面 → 「SEO 設定」で以下を設定：

| 項目 | 設定値 |
|------|--------|
| ページタイトル | こころのトーク ～いつもそばにいるよ～ \| AI メンタルサポート |
| メタディスクリプション | LINEで使える日本語専用AIメンタルサポート。24時間365日、心の対話をお届けします。月額980円〜。 |
| OGP 画像 | サービスのアイキャッチ画像（1200×630px 推奨） |

- [ ] Google Search Console にログイン・プロパティ作成済み

## よくあるトラブル

### Q. 所有権確認が失敗する
**A.** ペライチのページが「公開済み」になっているか確認してください。下書き状態ではメタタグが反映されません。

### Q. サイトマップが「取得できませんでした」になる
**A.** ファイルパスが正しいか確認。ペライチの場合、独自ドメインプランへのアップグレードが必要な場合があります。

### Q. インデックスされるまで時間がかかる
**A.** 新しいサイトは 1〜4 週間かかることがあります。定期的にコンテンツを更新することでクロール頻度が上がります。
